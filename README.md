# To-Do App 

The Todo App is a simple application built with React that allows users to manage their tasks efficiently and stay organized. It provides a user-friendly interface for adding new tasks to a todo list and deleting completed tasks.

## Features

- Add Tasks: Users can enter new tasks in the input field provided and add them to their todo list.
- Delete Tasks: Completed tasks can be easily deleted by clicking on the task item.

## Getting Started

To use the app, visit the website at https://sreeyu.github.io/todo_app/ .The website is accessible from any modern web browser.

## Usage
- Access the [Todo App](https://sreeyu.github.io/todo_app/) application on your web browser.
- Enter a task in the input field provided and press the "Add Task" button or press Enter to add the task to your todo list.
- Each task will be displayed as an item in the list.
- To delete a task, simply click on the task item, and it will be removed from your todo list.
